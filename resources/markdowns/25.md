# 02 资金通讯类
## 2.1 现金
**港币现金**在香港依然是必要的，因此建议来港新生事先准备一些，以免遇到一些突发情况手忙脚乱。

-   如果通过福田口岸/罗湖口岸/西九龙高铁过关，在过关后的落马洲/罗湖/柯士甸港铁站即可**购买****[八达通](https://mp.weixin.qq.com/s?__biz=MzAwNjAxNTYxNw==&mid=2651458956&idx=1&sn=9cc47e92d78d66479b605edad00b240e&scene=21#wechat_redirect)**乘坐公共交通，此时通常需要现金支付150港币，其中包含押金50港币和卡内100港币余额。
    
-   如果是打车，香港有**许多的士目前依然只接受现金**付款（支持微信、支付宝、八达通的的士会在窗户上贴上相应标志），从深圳湾口岸/福田口岸/罗湖口岸搭的士前往科大致需要300-500港币。

## 2.2 银行卡
**内地的银行卡**在香港也是可以使用的，香港支持银行卡的店面会注明是否支持银联、Visa和MasterCard卡，此时刷内地的银行卡会直接按照实时汇率扣除费用。

大家也可以办理一张**香港的银行卡**，以便在港生活，**[MSSS今年与恒生银行有合作开户优惠](https://mp.weixin.qq.com/s?__biz=MzAwNjAxNTYxNw==&mid=2651458680&idx=1&sn=1472f5417686ed3330670c0ec8228d12&scene=21#wechat_redirect)**，科大目前校内有恒生银行和中国银行（香港）的分行和ATM机。

## 2.3 电话卡

**内地的电话卡**在开通漫游的情况下，在香港也是可以正常使用（打电话、移动数据上网）的，但是**资费可能偏高**。另外，在香港注册一些APP或者活动报名时，电话号码可能只能填写8位数的香港手机号。因此我们强烈建议大家办一张香港的电话卡。

此前我们已经发布了**[香港电话卡攻略](https://mp.weixin.qq.com/s?__biz=MzAwNjAxNTYxNw==&mid=2651457414&idx=1&sn=60053ba3b8c13da1ef0d57a963857644&scene=21#wechat_redirect)**，如果还没有考虑好是否要**上台（办理月费套餐）**，可以先在正规平台购买一张**储蓄卡**使用先。想要上台的同学可以参考**[MSSS今年与CSL公司合作推出的上台计划](https://mp.weixin.qq.com/s?__biz=MzAwNjAxNTYxNw==&mid=2651457441&idx=1&sn=a739e71ebc1999973d309c37b385fd5f&scene=21#wechat_redirect)**。

  
# 03 生活物品类

## 3.1 衣物及床上用品

**常服****：**香港春夏多炎热，夏季温度可达30多度，夏装穿的时间比较长。但需要注意的是，夏天香港室内的冷气相对比较足，所以也需要准备轻便的外衣。由于科大在海边山上，秋冬会更湿冷一些，如果比较怕冷的话可以准备一两套厚衣服。大家可以到**香港天文台**查找分区气温（下图为香港天文台整理的香港平均气温和湿度），根据自己的耐热、耐湿能力来准备衣服。

香港天文台 https://www.hko.gov.hk/

**正装****：**建议提前与本专业的学长学姐交流一下在学校学习生活中穿正装的频率。一般来说，参加大型仪式活动活动、舞会等场合会用到，商科同学的面试、行业实习或做汇报时也可能用到。如果暂时还拿不准，也可以不用准备，在香港需要用时再买。  

**起居服****：**建议带一两套睡衣。

**鞋子****：**如运动鞋、休闲鞋、拖鞋等。

**其他特殊场合的衣服****：**如运动服、游泳装备等。

**床上用品****：**香港的单人床尺寸一般是80～100cm，可自带两套换洗也可抵港后购买，一般在日本城、宜家等商店进行购买。如果实在害怕席地而睡（有的出租房没有床垫），可以带一条薄被和被单。

## 3.2 个人护理及药品

**日化用品****：**护肤品、洗漱用品、化妆品等，可以根据个人需求来携带，也可以考虑到香港之后再购买。香港这边的日化用品普遍还是比较便宜的，学校附近的坑口、宝林均有sasa、万宁之类的连锁店。

**药品****：**如果自己有比较常用的药可以带上，如过敏药、胃药等等，非管制类药品是可以过海关的。如果是处方药的话，建议也把医生处方携带上。药品的量保证自用的水平，不要带太多。如果不带的话也没关系，虽然香港的公立医院很慢、私立医院很贵，但是科大校诊所可以满足大家绝大部分的需求，看小病和开三天以下的药是免费的；另外在商场中也有标有红十字logo的药房可以买药；此外科大也为非本地生购买了医保，详情可在MSSS微信公众号首页回复“看病”查看攻略。

请大家务必注意，**确保自己所带物品中都是符合香港入境规定**的，可以在以下香港海关网站上进行查询。不要冒险携带违禁物品，如果被查出，可能会被仔细审查盘问耽误过关时间，更可能留下记录导致每次过关都要进小黑屋。

## 3.3 学习用品

**学习用品****：**香港文具卖得稍贵，可以根据个人学习习惯自带铅笔、中性笔、圆珠笔、笔记本、草稿纸、移动硬盘等。有计算器的话也可以带来香港，但是需要注意，科大授课老师一般会要求考试中使用的计算器需要**符合香港考评局要求**，即上面有H.K.E.A.A. APPROVED字样，大家也可以来了香港再借/买。

**电脑、平板**：在香港购买电脑和平板如果按官网价格，一般会比内地便宜，但是内地的淘宝、京东、拼多多等各种平台又要更便宜一些。许多同学想趁新学期购置新电脑和平板，我们也为大家整理了**教育优惠和学生机**的攻略，这两天我们也会同步发出。

## 3.4 其他

**生活快消品**：如纸巾、洗衣液等，建议仅带短期够用的量就可以啦，到港后可以到超市（百佳、惠康等）购买，也可以在京东、天猫超市等平台网购， 一般会有满额包邮到港的活动，大家可以多蹲蹲。

**充电设备**：香港插头规格是英国标准的，要带转换插头，建议可以再带个英标转大陆标的插线板。另外还有各种充电器、充电宝等。

**小家电**：如烧水壶、吹风机等，这些可以带来也可以在丰泽等超市、HKTV Mall网站上进行购买。另外科大附近普遍比较潮湿，学校宿舍的底楼空气湿度可达90%以上，不习惯的同学也需要购置抽湿机。这样大一些的家电带过来就不太方便了，可以考虑在港购买或收二手。可以多关注我们的二手群。
