# 01 介绍

新学年就要到来，同学们是否想为自己的学习用品添砖加瓦、更新换代呢？一直以来新生群里都在聊**学生机、教育优惠、返校季**，那么这些东西分别是什么呢？怎样才能更实惠地买到心仪的电脑呢？小编今天就来为大家一探究竟。

# 02 学生机
学生机是**「大学生电脑优惠计划」**的俗称，即一些公司或商家和香港诸多院校合作提供的优惠计划。参与到这个计划中的有多个港校，每个港校负责联络的单位、活动名称、品牌型号都可能不一样。目前我们主要知道**两个计划**，即

-   University Notebook Ownership Program 大学手提电脑优惠
    
-   U-Mac Program
    

科大这边负责联络的是科大学生会，活动名为**「香港科技大学学生会电脑节 HKUSTSU Electronics Fiesta」**，已经包含了以上两种品牌优惠，主页见下：

# 03 教育优惠

教育优惠则是各大电脑厂商直接针对各校学生的优惠，即各大厂商直接在自己的官网宣布的，持有有效学生证即可参与。以苹果为例，可以分为常规的**教育优惠和返校优惠**。

## 3.1 苹果教育优惠
这是全年都有的优惠。可以于**线上购买**，登入以下教育商店购买，后期需要上传一些核实学生身份的证明。还可以学生本人去香港**线下门店**，出示实体学生证即可。

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpvpwuPwViajGEc9uzGPiboPWNm55NCIDLn00ymeTv5kUgibqTYgcib1P1UuianEy7LZts2RD4W6Na3Wqdw/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=7)

https://www.apple.com/hk-zh-edu/store

## 3.2 返校优惠购买方式
苹果还有一项Back to School专上教育优惠，为了与上述教育优惠区分开，我们直接把Back to School翻译为返校优惠。返校优惠有**时间限制**，活动时间为**2025年****7月10日至9月30日**，购买**iPad/Mac**等产品以后，可以以优惠价购买**Apple Pencil/Airpods**等推广产品。大家可以在以下网站查询返校优惠的细则：
