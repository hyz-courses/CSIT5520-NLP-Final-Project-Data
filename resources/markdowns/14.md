# 01 香港本地购物

## 1.1 官方店铺购买

从**官方店铺**购买肯定是最保险的。一般来说，从官网上可以找到位于这家公司在香港的旗舰店、官方零售店以及相应商品的价格；部分还支持官网下单运送到家，比如苹果、宜家等等。

## 1.2 零售店

许多品牌会把自己的产品**授权给一些零售店**来进行售卖。比如说在**丰泽**可以找到飞利浦的电动牙刷、小米的空气净化器和戴尔的电脑等；在**莎莎**可以找到各种品牌的美妆产品。

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpshLg2GyvrxOvl6BvpYWQrkRWIET4Uokd70ou9onA7fgMApWAvfcWyrLOVNibw2ODp5CH6FosUDuUQ/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=2)

## 1.3 格价网站

所谓**格价网站**，就是把各种类型的商品及资讯都整理好供大家挑选对比。以**香港格价网（HK Price）**为例，香港很多实体电子产品店都会在这个网站上开设网店，大家可以挑选完成以后通过线上或者线下去店铺购买。除此之外网购还可以通过**HKTV mall**，**香港消费者委员会**也有一个官方的格价平台。

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpshLg2GyvrxOvl6BvpYWQrk73HkXKCQXB15DYb3ghSaVxVJV48LCmenUFUYeaNX8nsbO0ekogrdOg/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=3)

## 1.4 二手交易平台

**二手交易**想必很好理解，香港本地Facebook各群组、carousell网站等，MSSS也建有科大的二手群。这里要提醒大家要注意资金和人身安全，与陌生人交易时要**注意验证身份**，交易地点要选在**安全的场所**（比如科大校园内），谨防上当受骗。

# 02 从内地到香港的网购平台及服务
## 2.1 淘宝

在香港也是可以使用淘宝的，当你的地址选在香港时，自然就会需要选择跨境段送货方式。如图所示有**直运和集运**之分，这里我们依次介绍一下。

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpshLg2GyvrxOvl6BvpYWQrkjYgokk9hK2yWeIxp4m2eeAlIRs3eplcJPYsiaolbpNSlyADdpic1z0kw/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=5)

「菜鸟官方直运」和 「菜鸟官方直运—快捷」

  

这两种方式的特点是：

-   下单时地址**直接填香港地址**，可以看到商品费用和相应的运费。在你当次购买的所有东西到齐之后自动合单，随后运来香港，无需别的操作。
    
-   **运费是根据快递的体积和重量**来计算的，因此大件家具不太划算，因为包裹还会限制长宽高，同时送上楼还需要额外付费，最后运费会比较贵，推荐集运（本文第三部分）或收二手（见本文1.4部分）。
    
-   常见的取件方式有**自取或者上门派送**，自提可以选一个距离住处比较近的自提点，自提比上门便宜。
    
-   「菜鸟官方直运—快捷」比「菜鸟官方直运」会**更贵一些**，并且是每个包裹分开计算重量、分开计价，最后合并。也就是说如果你买了一堆小件的话，可能会收若干个「首重」的钱，在十个店下单的十个袖扣就要175块钱运费了。
    

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpshLg2GyvrxOvl6BvpYWQrkZ8p54ichn2Om1bg0LFZTjjicO1Qgkyct3U3engGceFBDvNvgxjv5PvdA/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=6)

「菜鸟官方集运—标准」和「菜鸟官方集运—快捷」

这两种方式是淘宝合作的集运，特点是：

-   下单地址是香港地址，随后**自动生成一个集运仓地址**，店家会把商品寄到这个集运仓，这个时候你只需要付商品费用，不用付运费。
    
-   在你的所有商品都到集运仓之后，点击**「前往合单」**手动选择需要集运的东西并支付运费。这里「菜鸟官方集运—标准」和「菜鸟官方集运—快捷」的区别与直运一样。  
    

PS：有时大家可能还会看到一个叫做“联系卖家发货”的选项。如果你选择了这个选项，卖家会直接收到你的香港地址，然后由你们来协商怎么把这个东西发到你的香港地址。

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpshLg2GyvrxOvl6BvpYWQrkK5aswSicG2m3TAI7wibkrgT4RAkh54JnQ2FfwDiaydsy79nGYjibcfThAQ/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=7)

除了以上方式以外，淘宝的部分商品有**满99香港跨境段包邮的活动**，可以在满减页面挑选商品，这样也是直接到香港且无需额外运费的，详情可见以下淘宝官网页面。

## 2.2 拼多多

拼多多**官方也可以集运**，操作大致流程如下：

1.  选择香港的收货地址，选择官方集运；
    
2.  商品到达集运仓以后，在订单详细中选择去集运；
    
3.  选择收货方式：自提点/自提柜，送货上门；
    
4.  支付后会在订单详细中看到包裹情况。配送包括：神州/顺丰/极兔（价格不一样，可自行在app查看）
    
## 2.3 京东及天猫超市

# 3 转运服务

## 3.1 转运的业务逻辑

大家在实际购物可能会发现，有一些商品是没有办法直接到香港的，或者说淘宝京东自身的集运费用过高。那么此时，转运服务也就应运而生了。

以某个提供**「内地→香港转运服务」**的公司为例，其流程大致可以拆解为以下几个步骤：

1.  在这个公司的官网/APP/小程序上注册帐号，获得一个「内地仓库地址」，一般在广东，比如东莞、深圳等地。
    
2.  网购填地址时/朋友给你寄东西时，都寄到**上述的这个「内地仓库地址」**。
    
3.  公司会帮你把物品，从上述的「内地仓库地址」运到**公司指定的「香港取货点」**。
    
4.  你到香港这个取货点取货，你付的款项就是**从「内地仓库地址」到「香港取货点」这一段的费用**，一般根据物品的重量、体积来定。
    
## 3.2 科大同学常用的转运

常用的转运服务包括：漂流瓶、4px、淘集运、神州集运等等（MSSS并没有收推荐费！只是暂时想到这些，大家有感觉好用的转运商也欢迎**在评论区补充**）。

这些公司的**业务逻辑**都是一样的，只是**计价方式、取货地点、折扣优惠**等不一样，大家可以根据自己的实际需求来选择。
