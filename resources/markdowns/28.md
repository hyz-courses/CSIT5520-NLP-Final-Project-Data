# 01 介绍

**香港特别行政区大学图书馆长联席会**，英文为the Joint Universities Librarians Advisor Committee，简称JULAC，其中包含八所公立院校。

**JULAC Card**就是联席会提供给这八所公立院校的学生及职员的「联校图书证」，通过这张卡可以出入这些学校的图书馆，因此一般**俗称**「八校通」或「八校图书馆卡」。

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpty0VE9UztrEoticse2Zg0wPc41vS3lILDebtAvfticvn0bT0bcQAzGwSFENicdmKVcbcZz7vWuT84sA/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)

JULAC的八所成员高校

八校卡分为**Borrower card和Reader card**两种：

1.  Borrower Card面向的是**联校的全职研究/行政人员和研究生**，有效期一般是到预计毕业时间，申领Borrower card不需要申请费或激活费，如果要补卡的话需要交50港币；
    
2.  Reader Card面向的是**联校的本科生**，Reader card每次申请有效期为一年（当年8月1日至次年7月31日），每在一个学校激活都需要交50港币，补卡需要交50港币。
    
# 02 申请及领取

以下内容以**申请Borrower card**为例。申请Reader card的步骤大同小异。

1. 进入科大图书馆的**JULAC卡申请页面**，点击"online form"

以上说的是**在线申请**，当然其实也可以直接前往G层Circulation Counter（刚进图书馆闸门的左边柜台）提交资料申请。由于在线申请的手续非常简单，可能并不会有很多人首次就去线下柜台办理。

然而如果JULAC卡丢失，进入网页想再申请一次，就会发现网页上写着“You are existing JULAC Card user. Please go to circulation counter.”，也就是**必须去Circulation Counter进行补办**啦。

补办的手续也非常简单，职员调取你的资料以后，现场重拍一张照片，新的JULAC卡当场就可以做好。

# 03 使用指南

## 3.1 登记

获得八校卡以后，并不是可以直接刷卡进入其他七所大学的图书馆，当你初次进入某校图书馆时，还需要在该校图书馆前台**进行登记**。

你应该把你的科大卡、八校卡都交给该校图书馆前台，他们会给你一个可以在该校图书馆电脑上登录的**访客帐号和密码**，并登记这张八校卡，然后让你进入图书馆。等登记结束（大概一个工作日）以后，你就可以使用八校卡**直接刷卡过闸机进入图书馆**。

## 3.2 繁忙时间

每个学校到期末考试周**繁忙时期**，图书馆可能都会**限制JULAC卡持有人的进入时间**，把使用权优先留给本校学生职员。大家会收到邮件提醒，当然也可以在前往某校图书馆之前查看**该校图书馆官方网站**的说明。

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpty0VE9UztrEoticse2Zg0wPmEJUibbaicnqKzqKZkHLrrl4kxYGTic18GJcd4OqGX9ND3LwLowpicQCmQ/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=7)

由于在城大图书馆登记过，因此收到城大图书馆的邮件

## 3.3 进入学校
JULAC卡让大家可以进入八校图书馆，这是由大学图书馆长联席会决定的，而**校园的进入**则一般是由各校校园管理处/安保团队来负责的。从常规理性而言，既然JULAC卡可以让你进入图书馆，自然也应该能进入校园，事实也的确如此，不过少数学校目前还不是完全开放，可能需要少量信息登记，大家可以提前留出时间。

目前八校对**个人访客（不开车、人比较少）**的限制是：

-   港大：游客进入港大需要预约，但是持JULAC卡的其他港校同学可以随港大同学一起进入校园，被保安抽查时出示JULAC卡即可
    
-   中大：现场凭身份证件登记即可
    
-   科大：直接进入
    
-   城大：一般访客需要提前登记获得QR code，JULAC卡持有人可以在现场使用JULAC卡进行登记获得访客二维码进入
    
-   理大：一般访客需要提前登记获得QR code，JULAC卡持有人可以在现场使用JULAC卡登记获得访客二维码进入
    
-   浸会：直接进入
    
-   教大：现场凭身份证件登记即可
    
-   岭南：直接进入
    

## 3.4 借书
Reader Card和Borrower Card的区别从字面意思上就能看出来，Borrower Card可以借书。

大家可以在**Hong Kong Academic Library Link(HKALL)**上选择借阅其他学校图书馆的藏书；之后藏书会**送到科大的图书馆**，因此只要在科大图书馆借/还即可。
