# 1 校内遗失物品

## 1.1 常见寻找思路

无论大家在校内哪里丢失东西，请记住就近原则，第一时间回丢失物品的**建筑的管理处**找，即：

-   在图书馆丢失东西 → 去图书馆**前台**
    
-   在LG1餐厅丢失东西 → 去LG1餐厅**前台**
    
-   在LG1羽毛球场丢失东西 → 去LG1运动场**前台**
    
-   在宿舍大门附近或走廊遗落东西 → 去宿舍**前台**
    

如果是在**教室**丢失东西，每个教室前方的**讲台下面**都有装着疑似丢失物品的盒子，可以优先检查。

如果上述地方没有找到，或者你完全想不起来可能是在哪里丢的，那么就去**保安中心 Security Center**。

## 1.2 保安中心位置

在今年以前，保安中心位置就在火鸟附近，不过从11月8日开始，保安中心的位置已经改为**LG7353**。具体前往方式可见下两图，学校推荐三种路线：

1.  乘10-12号电梯（图书馆附近，前往LG7餐厅与Fusion）前往LG7，穿过卸货区可达；
    
2.  乘15号电梯（南北小厨旁边）前往LG7，左转后直行可达。
    
3.  乘3号电梯（星巴克附近）前往LG7，右转后直行可达。

去保安中心报告丢失的物品详情，会有两种情况：

-   如果已经有人送回来了，职员会和你核对一下物品资料（比如问你丢失的物品是什么东西，什么颜色等等）然后交给你，你需要填一页已经取回物品的文件；
    
-   如果暂时还没有人送回来，职员会让你先填“遗失物品”的一页文件，留下你的联系电话及个人信息，这样之后如果有人送回保安中心，保安中心就会通知你。
    

保安中心的联络电话为 **2358 6565**。

## 1.3 科大证件补办

如果一些校内证件（如学生卡等）遗失后大家真的怎么也找不到，那只能选择到学校补办。这里我们补充一下补办证件的款额与相关办公室：

-   **学生证**：补办需要**150**港币，扫下方二维码即可打开网站，用学生账号登陆后直接填写表格申请新卡。

补办学生卡：https://registry.hkust.edu.hk/resource-library/lossreplacement-hkust-card

  

-   **宿舍房卡**：补办需要**100**港币，联系你的宿舍前台即可。
    

此外，入住宿舍时交给我们的**卡套**也是不能丢或者损坏的，在退宿时需要原封不动归还，若损失也要交**10**港币。

# 2 校内捡到物品

## 2.1 交回管理处

若是我们在校园内捡到了物品但不知道归还给谁，总体的步骤与上方遗失物品是一样的：

-   如果比较赶时间的话，就近交给相应**建筑的管理处**即可（在图书馆就给图书馆前台，在餐厅就给餐厅前台，等等）；
    
-   如果是在路上、教室等没有明显建筑物管理处的地方捡到的，那么就交到**保安中心 LG7353**。
    

## 2.2 给同学发邮件

如果该物品上有**失主的姓名**（比如学生卡，学生八达通等），你可以通过**Student Portal**网站右侧的搜索框搜索到该同学或职员的邮箱（见下图），直接发邮件提醒他去领取。

# 3 校外遗失物品

## 3.1 公共交通

除了在学校内遗失物品，在各类交通工具上遗失物品也很常见，尤其是我们的11小巴。这一部分里，我们讲一下各类交通工具的联系方式，包括小巴，大巴和地铁。

-   **小巴**：如果大家有物品遗失，请直接**电话联系公司**，然后公司会联系你乘坐的车次帮忙寻找，在找到后会打电话给你让你去公司取。由于香港经营小巴的公司数量很多，这里我们以科大人必乘坐的**11/11M**举例，11路小巴的联系电话为**27196669 / 27196668**。公司名字叫**喜龙有限公司**，地图可导航。  
    如果你不确定你乘坐的小巴的公司电话，可以在原地等待至下一辆**相同车次**的小巴，询问车上司机，他会给你该车次线路负责人的联系方式。
    
-   **九巴 （KMB）**：即红色巴士。直接去**九巴官网或九巴app**进行报失（见下图），在bot1933程序中填写巴士号，上下车时间，方向等。九巴的联系电话为**37081333**，在备案后可以拨打电话询问进度，找到后会短信通知领取地点。

## 3.2 商场及购物中心

在购物中心遗失个人物品，前往购物中心的**管理处或客服中心**即可。以坑口东港城为例，直接来到2楼的**顾客服务中心**（Customer Care Center）说明情况并报告信息即可，商场会帮你查询或广播。

如果仍然找不到，请立刻前往最近的**警署报案**挂失。此外，警署也有电子报案中心可线上挂失（网址见下图），警方会在系统里帮你查询，如有人捡到上交，警署会联系大家。

## 3.3 香港证件挂失补办

如果在港不幸丢失了证件（香港身份证/港澳通行证），一定要**首先前往**最近的警署或是网上**报案挂失**（与3.2报案方式一致），一是方便别人上交后联系，二是补办证件需要出示。以下是补办香港身份证与港澳通行证的基本流程：

-   **香港身份证补办：无需预约**，可以直接前往入境事务处补办，带好警署开示的**丢失证明**与**学生签**（小白条）即可补办，缴费**370港币**拿到行街纸，10个工作日后可用行街纸领取身份证。
    
-   **港澳通行证补办：**需要在**香港中旅社**网站预约（网址见下图）。办理当天准备好内地身份证原件与复印件、丢失证明、小白条与evisa，以及蓝底照片（当场照需要50港币），10个工作日后可取新的证件。
