# 01 常见术语

## 1.1 储值卡

**储值卡**是一种预先支付费用的电话卡，用于支付电话服务费用，例如国际长途或本地通话费用。用户购买卡后，卡内会**预存一定的金额**，用完后可以**充值或丢弃**。﻿

储值卡可以在淘宝或者香港的各种**便利店或超市**买到。储值卡有一定使用期限，可以通过充值来延长使用期限。费用通常是按拨打的通话时间、短信、上网时间及流量计算，直接在卡的余额里扣除。

优点

1.  无合约绑定，停机甚至都不用申请，直接不用就完事儿了；
    
2.  对于流量需求不大的同学来说价格便宜。
    

缺点

1.  信号没有上台卡好且稳定性不高；
    
2.  超出范围内的流量费用较高，对上网需求大的同学而言不划算；
    
3.  储值卡的号码是循环使用的，因此可能会出现自己新买的电话卡已经在某些App上已经被绑定了之类的问题（当然了现在号码是属于你的，你可以收验证码重新绑定，只是会需要一些手续）；
    
4.  如果有一些粗心的同学忘记在储值卡有效期内给它充值的话可能会出现储值卡过期了然后被迫重新绑定各种App或者服务的情况。
    

## 1.2 上台卡

**上台卡**指的是后付费SIM卡，也就是需要**签约套餐**并按月付费的手机卡。也就是说，上台可以理解成运营商的包月套餐服务。通常合约期为一年或两年，费用月结。

优点

1.  流量价格相对便宜；
    
2.  信号稳定，网速较快；
    
3.  电话号被绑定之类的事情发生概率较低。
    

缺点

1.  不灵活，如果要解约必须要提前申请，否则会自动延续；
    
2.  每个月有比较高的月费，对于流量和电话用量不大的同学来说有点贵。
    

## 1.3 携号上台 / 转台

综合对比上述**1.1储值卡**和**1.2上台卡**的优缺点，两者最大的区别其实就是在**流量的价格**上。因此小编认为上台更适合经常出门的同学，而储值卡更适合活动范围主要是学校和宿舍等地的同学。

这里小编建议大家可以先用一段时间储值卡，如果之后需要上台的话，则可以**「携号上台」**，即**保留目前储值卡的这个号码**签约上台。方式是拿着SIM卡的卡套和其他上台所需材料，前往营业厅或在对应网上营业厅进行办理。

需要注意，每个运营商的话费和流量价格也是不一样的，如果你用了A公司的储值卡，想办B公司的月费套餐（上台），也是可以的，不需要换号码，在香港称之为**「携号转台」**。方法也是带着手机卡和证件去营业厅办理，或在对应的网上营业厅进行办理。

# 02 电话卡实名登记

香港的《电讯(登记用户识别卡)规例》已于2021年9月1日正式生效以实施电话智能卡实名登记制，也就是俗称的**「电话卡实名登记制」**。原有电话储值卡如在**2023年2月23日限期**前未完成登记则不能继续使用；新购买的储值卡都必须在实名登记以后才能使用。

如果大家发现自己通过某种渠道购买的电话卡，还没进行实名制登记就能使用了，那就要小心了，这可能是已经被别人登记过的**「二手卡」**。

# 04 广告电话与诈骗电话

## 4.1 广告电话

使用香港电话卡收到销售广告电话可以说是普遍现象。五花八门的推销手法，与内地有异曲同工之妙，优惠多多，陷阱也多多。**常见的广告推销**包括：强积金、低息贷款、银行卡等等。

在这里我们提醒大家：

1.  办理银行卡业务找正规官方途径就好；
    
2.  在香港，非本地学生一般是不能进行银行贷款的；
    
3.  强积金不针对没有工资的人士，请注意RPG同学领取的研究生助学金PGS在法理上不是工资，因此并不适用。
    

我们建议大家：

1.  收到陌生电话时，**先等对面开腔**，如果发现是无用的广告推销或者电话录音，**直接挂断**即可；
    
2.  不要和无用的广告商尬聊，这可能导致你的电话卡被标记为「活跃用户」，增大此后再次接受广告的概率；
    

当然，大家可能有疑问，如果拒接陌生电话，是否可能错过学校部门的重要电话？不用太担心，科大座机电话专属**2358或3469开头**，很容易辨别；而且部门一般是发邮件联系你，如果真的致电联络，也会**直接开口问「请问是否是XX同学」**，很容易区分。

## 4.2 诈骗电话

香港电话诈骗相当猖獗，可以说几乎每个人都会接到不止一个诈骗电话，这里我们先分享一些**最常见的电话诈骗**手法：

1.  骗徒（3-5人以团伙作案），冒充不同的角色，例如电讯公司职员、入境处职员、内地公安、内地司法机构等；
    
2.  一般以**电话录音、视频形式**；
    
3.  号称涉及内地刑事罪行，要求核对**个人资料**；
    
4.  「恐吓」受害者限制**出入境自由**；
    
5.  要求提供个人**银行**资料与**资金**来源；
    
6.  出示内地执法部门的假公安证明文件及假法庭通缉令。

# 05 宽频服务

想必大家在租房的时候都有过疑问，那就是**房屋内的无线网络**怎样置办？

香港的电讯服务供应商，很多是同时运营**固网**、**宽频**、**流动通讯**及**媒体娱乐服务**的，这里我们略作介绍：

1.  **固网**，即固网电话，是通过金属线或光纤线等固态媒体传送讯号的电信网络，绝大多数人在自己家已经不用固定电话了，香港的办公室里则会有，这里略过；
    
2.  **宽频**，即内地说的宽带，宽带是一种相对的描述方式，频率的范围愈大，也就是带宽愈高时，能够发送的数据也相对增加。简单地说，你需要确保房间内的宽频已经配置好了，这样就有**网线**可以接到电脑上进行上网；然后你再买个**路由器**，就可以配置好房间内的无线上网了；
    
3.  **流动通讯**，这说的就是移动电话通信了，也就是本篇推文主要介绍的香港电话卡；
    
4.  **媒体娱乐服务**，这说的是IPTV / OTT视频 / 音乐等订阅，不过绝大多数同学也并不会在自己家买个电视机看香港节目，所以也略过。
    

这里小编还提醒大家一下，由于一家电讯服务供应商运营以上多个服务，所以有时它们推出的套餐内，把**宽频和电话卡打包购买**，也许还有**额外的折扣**。大家在选购时，也可以把这一点考虑在内。

# 06 温馨提示

1\. 香港的通信市场采用的是**全市场服务制度**，除了我们前面说的几大运营商之外还有一些比较小的运营商，这些运营商的**服务保障和售后**都不算好，并不建议大家购买这些运营商的储值卡或者办理上台。此外，在**淘宝**买电话卡如果遇到一些纠纷也**不方便维权**。我们更建议的还是在**官方途径**或者**大超市/连锁超市**购买我们前面提到的大运营商的储值卡或者上台。  

2\. 储值卡与上台（月费套餐）的优缺点在本文亦有介绍，市面上的**评测**也很多，当然大多也有**利益相关**，大家可以多研究多讨论。

3\. 在香港如果不买香港的手机卡，**坚持使用内地手机号**，其实也是一种选择，但需要考虑计费的问题，以及部分香港软件是必须香港手机号登记的。**根据经验**，绝大部分同学还是会购买使用香港手机号的，这期间可以把内地手机号的套餐全部取消，**保持最低限度的资费和使用**，毕竟偶尔还要收验证码。因此如果是支持双卡的手机会更方便。
