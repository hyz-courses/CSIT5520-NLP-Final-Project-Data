# 02 校历解读

在校历上，你可以方便地查看**公众假期**的具体安排、**Add&Drop Period**的时间、RPG学生**最迟提交thesis**的时间等等，可以说是非常重要了。以下画一下重点：

-   **2025年7月15日（即明天）**是研究生院给出的**[寄送学务资料](https://mp.weixin.qq.com/s?__biz=MzAwNjAxNTYxNw==&mid=2651455453&idx=1&sn=c259e3ee76135f0b808b2bc22e83517e&scene=21#wechat_redirect)**的截止日期，如果大家还没来得及寄送，可以给研究生院写邮件申请延迟ddl；
    
-   完成学务资料验证之后，等候7月末**[线上注册](https://mp.weixin.qq.com/s?__biz=MzAwNjAxNTYxNw==&mid=2651447345&idx=1&sn=c6e6f02aaeb4a6e9a562b491a72af402&scene=21#wechat_redirect)**即可；
    
-   完成注册以后，**8月26日**起可以开始**[进行选课](https://mp.weixin.qq.com/s?__biz=MzAwNjAxNTYxNw==&mid=2651448659&idx=1&sn=efe5200d5aa937f9032d2c8ffa9a32a2&scene=21#wechat_redirect)**；
    
-   **2025年9月1日**，秋季学期开学，一般开学的第二天为学费due day。一般来说，如果项目组**没有发自己的安排**，那么默认按照校历来开学；反之则以**项目组的安排为准**，你可能提前上课；
    
-   **2025年9月1日至13日**是**Add&Drop Period**。它的意思是：这段时间你可以自由add或者drop课，你可以先选一些感兴趣的课，在这两周期间都去上一下看一下，再决定最终上哪些课；
    
-   **2025年9月15日~27日**是**Extended Drop Period - for PG courses only**。它的意思是：这段时间你可以自由drop一些PG课程；
    
-   **2025年12月19日**，秋季学期的所有考试全部结束；
    
-   **2026年2月2日（腊月十五）**春季学期开学，2月17日（正月初一，星期二）到2月19日（正月初三，星期四）这三天为香港的公众假期，也对应春季学期的Week3；
    
-   **2026年5月29日**，春季学期结束。
    

红色的字样（一些还以黄色背景标注）的是**公众假期**。关于公众假期，我们会在下一节作出详细解释。

# 03 公众假期说明

香港的“假期”其实是有两个概念的，分别是**公众假期和法定假日**。

**公众假期**，General/Public Holidays，由《公众假期条例》所规定，政府部门、银行、教育机构等一般会在公众假期放假。所有的星期日都是公众假期。

**法定假日**，Statutory Holidays，由《雇佣条例》所固定，雇主一般让员工在法定假日放假。

校历中的Public Holiday（红色与黄色）即为**公众假期**。2025年中，法定假日有14天，公众假日则是17天加上所有的星期日。**法定假日的14天全部都是公众假期**。

科大算作教育机构，所以上述的**公众假期**都是遵守的，各**院系办公室**等等在公众假期都是放假的，所以公众假期行政办公室**很少会回复邮件**。但是大家如果仔细查看上述《公众假期条件》和《雇佣条例》就会发现，公众假期和法定假日在某些情况下**也是可以上班**的。具体到科大的话，保安中心很显然就是全勤的，而且**有的老师还会安排上课，导师也会安排组会**，大家最好还是留意**邮件**及**canvas**：）

一般而言，对**UG和TPG**同学而言，不上课就可以理解成假期，确切点就是：

-   星期天和公众假期是不安排课堂的，算**假期**；
    
-   选什么课是自己决定的，如果能排出来周一到周五的某天是没有课的，一般称之为**day off**；
    
-   不计attendance的课，可以不去，可以理解成**“只要胆子大，天天都是假”**；
    
-   Mid-term break和Study Break也不安排课，可以理解成**复习周**；
    
-   秋季学期考试全部结束以后就是假期，可以理解成**寒假**；
    
-   春季学期考试全部结束以后就是假期，可以理解成**暑假**。
    

对**RPG**同学而言，上课已经不是关键了，学习生活相当于每个月领研究生助学金(PGS)或其他奖学金作为“工资”进行研究工作，因此放假情况其实跟职员有些像，即**公众假期（包括星期天）**放假，然后每学年年假为**21天**（需要找院系签字通过）。

如果严格按照**香港的过年3天公众假期（初一到初三）**来放假，其他日期一天也不请假的话，那么同学就需要在**2026年2月16日星期一（腊月廿九）**上完课然后晚上回家，正月初一到正月初三拜年，正月初三晚上回到香港，准备**2月20日星期五（正月初四）**继续上课。

这显然是非常不合情理的，估计大量内地学生都会多请几天假，比如只要把**周一2月16日、周五2月20日请假**，就可以凑出一周九天假期了——正常来说，可能也不需要请假，因为**各位老师也是需要回家过年**的，周一和周五的授课老师想必都会提前发调课通知。
