# 02 运动技能课程

## 2.1 课程介绍

**运动技能课程**是对**Sports Skill Classes**的直译，方便大家理解。这类课程是在秋季、春季、夏季学期举办的，每门课包含8节课（每节课1.5小时），大概从第四周开始上课。

**本科生**需要完成至少9小时的运动技能课程（也就是说8节课里至少要去6次），才能满足HMAW1905的要求，而HMAW1905是本科生的毕业要求之一。**研究生**没有HMAW1905这门课，毕业也没有任何运动技能课程的要求，因此不占学分、缺席也不会有惩罚（不过建议不要缺席太多，以免浪费资源）。

**2025-26学年秋季开设的课程**包括以下6大类40种，每一种课程点击以后可以看到相应的班次的上课时间、上课地点、授课老师等信息。

## 2.2 选课规则与时间线

由于UG学生需要选运动技能课程才能毕业，PG学生无此要求，因此**UG学生选课优先级更高**。又因为UG学生也很多，**学校再次分流**，让商学院、人文社科、跨学科学院三个学院的同学在秋季选课优先级更高，让理学院、工学院两个学院的同学在春季选课优先级更高，即：

-   **2025-26秋季学期选课优先级**：SBM/SHSS/AIS的本科生 > SSCI/SENG的本科生 > 研究生；
    
-   **2025-26春季学期选课优先级**：SSCI/SENG的本科生 > SBM/SHSS/AIS的本科生 > 研究生；

# 03 水上活动课程

## 3.1 课程介绍

香港科技大学毗邻香港最佳水域之一。我们的校园非常适合开展**各种水上运动**，包括龙舟、沿海划艇、帆板、皮划艇、帆船等等。如果我们不采取措施，与海滨失去联系，**那将是多么可惜**！因此，校长决定充分利用我们令人羡慕的自然资源，全力支持在香港科技大学发展水上运动。（这段来自科大水上活动中心英语介绍的直译）

在水上活动中心的官网上，我们可以看到详细的介绍，包括过往的每一次课程通知。观察可以发现，基本上在**每年的5~11月**每个月都有课程，这些课程主要包括以下四种：

-   Kayaking 皮划艇
    
-   Standup Paddle 站立式划桨
    
-   Windsurfing 风帆
    
-   Dinghy Sailing 小艇帆船

## 3.2 选课方式

让我们以2025年8月18日发出的**2025年9月水上活动课程**举例，可以看到PDF的第一页有清晰的班次、时间、名额介绍，第二页有详细的选课时间线和机制介绍。这里画一下重点：

-   9月的水上课程，8月18日发出邮件、更新在官网上，8月21日开始报名，所以大家**不用担心看邮件看迟了**，只要你保证每天、每两天能看一次邮件，就不会错过；
    
-   报名成功的同学，需要**按时缴纳押金**，如果不能自己亲身前往，也可以授权其他同学代为缴纳，课程完成以后押金可以退回；
    
-   上这些水上活动课程需要有**能游50米的能力**（速度不限），在课程的最开始会在游泳池内进行这个测试；

# 04 运动类社团与校队

我们在此前给大家介绍过**科大的所有社团**（MSSS微信公众号首页回复“社团”可查看完整列表与介绍），这其中其实就包含一些**运动类社团**。此外，我们经常还能听到**“校队”**、“Uteam”的说法，运动类社团与校队的关系是怎样的呢？

在科大，共有超过30个Sports Club和超过50个Sports Team：

-   **Sports Club**是以**学生会属会**的形式进行注册，是一个完整的**社团**（有主席、内务副主席、外务副主席等十几个执委组成的内阁/庄），**社团的目的**就是开展相应的运动活动，促进该运动文化在科大的发展；
    
-   **Sports Team**就是在一项运动上科大的代表队了，其成员为**队长、副队长与队员**，会选拔队员代表科大去参与“大专杯”（可以理解成香港高校运动会）或者更高层次的运动比赛；

# 05 MSSS运动俱乐部

虽然学校已经有许多运动类社团与校队，但大家可能会发现其中local本科生比例比较高，或者作为会员/队员参与活动的门槛比较高；因此为了更好服务内地同学，我们MSSS也建立了**许多俱乐部**，其中也包括许多运动类俱乐部。

MSSS俱乐部不是一个在学校单独注册的社团，而是MSSS的部门/一部分，MSSS可以应需求在校内进行订场供俱乐部活动之用。在此前，我们已经进行过**俱乐部执委的招募**（负责管理俱乐部微信群、组织俱乐部活动），现在大家可以直接**加入俱乐部微信群**（也就是作为会员）参与活动啦！

![图片](https://mmbiz.qpic.cn/mmbiz_png/hXFlznfdEpu5Tp6ibVnUp5ZXwvRmOP6VkAem23TdicZoFzBemIFrnsg4aqVNb0lE3L06peZcIu50S9D4OCy1ayKw/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=14)

在MSSS微信公众号首页回复**“俱乐部”**查看加群方式

各俱乐部会举办对应的活动，比如户外俱乐部的**爬山露营活动**、健身俱乐部举办的**健身打卡活动**、舞蹈俱乐部举办的**随机舞蹈**、足球俱乐部的**日常踢球和友谊赛**、篮球俱乐部的校内**友谊赛和全港联赛**、乒乓球俱乐部的**锦标赛**、网球俱乐部的**网球教学课**，等等。

各俱乐部内也会**讨论交流相应运动**，比如说近期香港的赛事、接龙约人打球、运动装备购买等等；此外我们的许多俱乐部群内实际上也有一些科大校队成员，会为大家分享Sports Club的活动与Sports Team的选拔经验。MSSS始终保持着开放心态，欢迎各位内地同学**多多参与体育活动**，也为科大校队贡献更多内地生的力量！
