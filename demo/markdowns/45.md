# 02 顺丰寄送

## 2.1 大致流程

顺丰想必大家都不陌生，在香港的使用方法是类似的，大致流程如下：

1.  **填写寄件信息表**，可以在官网上/微信小程序里填写，或者在顺丰店拿一张寄件卡片扫描二维码填写；
    
2.  把**要寄的物品给顺丰店**或上门取件的顺丰小哥，他们会扫描拿到你的寄件信息，如果有填错的地方会帮你更改（比如你寄的东西是证件，物品类别却写的是行李衣服）；
    
3.  缴费，**完成寄送**，拿到寄送编号和二维码，随时查询进度；
    
4.  提醒**内地收货人上传身份信息**以签收物品。
    

## 2.2 官网信息

在香港使用顺丰，要确保搜索的是「顺丰香港」官网，小程序也可以从「順豊香港」微信公众号点击进入。

## 2.3 注意事项

这里小编结合自己的**使用体验**再来分享一些注意事项：

1.  科大校内下午会有顺丰小哥/大叔随机刷新，时间一般是1-2pm和5-6pm，地点是lift20一楼或lift22二楼。如果大家需要找他寄件，建议**早一点过来**，因为会有许多人排队取快递。
    
2.  找顺丰小哥/大叔寄件**仅限纸质文件**，大的物品是不行的，因为他不会带很多打包盒。
    
3.  从香港寄回内地，是需要**提供物品的价值**的（文件类不需要），也就是说需要给出相应发票或购买单据。很多人被这一关卡住，但其实**顺丰站**可以帮你**制作一个简单的发票**，你自己报一下这个物品价值多少就可以。这一点科大的顺丰小哥是做不了的。
    
4.  从香港寄回内地，一次只能寄**有限件数**（每样物品算一件），对价值也有上限要求。
    

# 03 香港邮政

## 3.1 大致流程

邮政可以寄信件，也可以**寄行李物品**，如果你的物品较多需要打包：

1.  可以先在网上填写相关资料，包括物品的长宽高和质量、收件人寄件人信息等，准备好纸箱，然后前往邮局；
    
2.  也可以直接前往邮政局购买纸箱，现场打包称重然后填写资料。

# 04 前往深圳寄送

## 4.1 过关路线
以上提到的两种方法都需要过海关，毕竟是从境外往境内寄东西。而涉及到海关检查，寄送的**时间**甚至**成功率**都是不太受自己控制的。那么有没有更快速、安心的方法呢？答案也是有的，而且非常简单，那就是**带着行李去深圳寄**。

从香港带东西去深圳，也要过海关，但是速度很快，而且只要是**自用的正常物品**基本都没问题。到了深圳以后，再往内地其他城市寄快递，可选的快递公司就很多了，费用、时间等等都更经济实惠。

那么怎样快速去深圳呢？科大同学比较常见的路线有三条：

-   科大坐11小巴去彩虹（住坑口、将军澳的同学上将军澳线然后转观塘线），随后观塘线坐到九龙塘站，转东铁线前往**罗湖站或落马洲站**，这两个站过关以后分别对应**深圳的罗湖站和福田口岸站**，附近都有许多快递寄送点。
    
-   乘坐中港通巴士（可以从坑口东港城等地出发），直接抵达**深圳湾口岸**，过关以后也能找到大量快递寄送点。
    
-   先通过地铁或巴士抵达西九龙高铁站，随后坐高铁抵达**深圳北站或福田站**。
    

除了以上路线以外，还可以通过**巴士/小巴的换乘**来省钱，比如说：

-   前往**深圳湾口岸**：B2线（元朗站出发，车费14.4港币），B3线（屯门码头出发，车费14.7港币），618小巴（天恩邨出发，车费14.4港币）；
    
-   前往**莲塘口岸**：B7线（上水出发，车费10港币），B8线（大围站，车费16.1港币），59S小巴（上水出发，车费9.9港币）；
